Setup

Up and running in 3 steps.

1
Install the module

Apps → search "Pan AI Pro" → Install

2
Add your API key

Settings → AI → "Use your own Anthropic account"

3
Select a Claude model

Create or edit an AI Agent → pick Claude from the dropdown

Or set the ODOO_AI_CLAUDE_TOKEN environment variable for headless deployments.
